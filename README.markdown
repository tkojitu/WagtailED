WagtailED
=========

WagtailED is a simple text editor for Android (4.0.3 or later).

Gestures
--------
The toggle button turns on Gesture Mode.

| Gesture             | Description           |
| ------------------- | --------------------- |
| Î›                  | Beginning of document |
| V                   | End of document       |
| fling left to right | tabify a line         |
| fling right to left | untabify a line       |
